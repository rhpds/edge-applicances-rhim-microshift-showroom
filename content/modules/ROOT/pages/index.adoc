= Building Self-contained Appliances with RHEL Image Mode and MicroShift

Welcome to this hands-on lab where you will learn how to build, deploy, and manage self-contained edge appliances using Red Hat Enterprise Linux (RHEL) Image Mode (`bootc`) and MicroShift.

image::redhat-image-mode.png[rhel-logo-black,55%,55%]

== Lab Overview

In this lab, you will progress through the entire lifecycle of an edge appliance:

* **Module 1: Foundation:** Understand the core concepts of RHEL Image Mode and the challenges of edge computing.
* **Module 2: Build:** Create a bootable appliance image that includes an embedded containerized application.
* **Module 3: Deploy:** Deploy your appliance to a virtual machine and verify its operation.
* **Module 4: Update:** Perform an atomic, image-based update to a new version of your appliance.
* **Module 5: Conclusion:** Recap the key takeaways and business value.

== Lab Access

The terminal window to your right is *already* logged into the lab environment. All steps of this lab are to be completed in this terminal.
